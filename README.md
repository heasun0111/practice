# 깃허브란?
## 깃허브는 2008년 설립된 ‘깃’(Git) 전문 호스팅 업체다.
## 깃허브의 영향력은 점점 커져 2015년 기준으로 깃허브 사용자는 1200만명이 넘었으며, 3030만개가 넘는 저장소가 운영되고 있다.
## 직원수는 400여명이다. 깃허브는 오픈소스 소프트웨어의 중심지(hub) 역할을 하면서 오픈소스 프로젝트가 널리 퍼지는 데 크게 기여하고 있다.
